# Valentines Proposal App

A playful and romantic Valentine web app built with Next.js.

## Features

- Mobile-first one-page proposal flow
- French/English language switch (French by default)
- Interactive "No" button that dodges pointer/touch
- Animated celebration page after "Yes"
- Footer credit with GitHub link
- Vercel Analytics integration for visitor tracking

## Tech Stack

- Next.js (App Router)
- TypeScript (strict)
- React

## Run Locally

```bash
npm install
npm run dev
```

Default local URL:

- `http://localhost:3001`

## Build

```bash
npm run build
npm run start
```
